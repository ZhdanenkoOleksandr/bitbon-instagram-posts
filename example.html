ℹ️ Системные требования

  • Браузер: Chrome, Edge, Safari (не требует установки)
  • Микрофон: Подключен и разрешен доступ
  • Язык: Русский (и другие поддерживаемые Web Speech API)
  • Интернет: Не требуется после первой загрузки

🔧 Технические детали

  • Использует встроенный Web Speech API браузера
  • Работает с input, textarea и contentEditable элементами
  • Вставляет текст в позицию курсора
  • Поддерживает промежуточные результаты распознавания
  • Совместим с React, Vue, Angular и другими фреймворками
